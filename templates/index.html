Find and organize hobby meetups

Hobby Meetups is a simple web application for browsing local activities, creating new meetup announcements, and joining other users' plans.

Meetup

A meetup contains a title, description, event time, location, and one or more categories. Logged-in users can create, edit, and delete the meetups they organize.

Join event

Users can join another person's meetup and leave a short comment for the organizer. Each meetup page shows the participant list and the comments attached to that meetup.

Search, categories, and profiles

Meetups can be searched with keywords such as title, location, description, or category. Each user also has a profile page with meetup statistics and a list of meetups they have created.
